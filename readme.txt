FM Radio for Arduino

See http://pc.watch.impress.co.jp/docs/column/musashino_proto/20090806_307235.html for detail (Japanese). A lot of photos and schematics are available.

FMradio.pde is the Arduino sketch running on Arduino Uno.
FMspectrum.pde is the Processing sketch to view spectrum chart of signals receiving.
